Class Detail
jsJango.JangoJobUtil(params, params)
JangoJobUtil provides a set of methods to manage Jango campaigns. You set the job id, make a call to one of the provided methods and *blamo* you have stopped/paused/resumed/resent a previous campaign.
Parameters:
{Obejct} params
default options to set.
{Number} params.jobId Optional, Default: -1
the jobId to set.
{Object} params
the default options to set on this Request.
{String} params.username Optional
the user name to login to jango with
{String} params.password Optional
password to login with
{Boolean} param.useSsl Optional, Default: true
set to false to not use SSL when connecting to jango.
{String} param.server Optional, Default: "api.jangomail.com"
the jango server to connect to.
{Integer} param.port Optional, Default: 443
the port to use when connecting to jango
{Object} param.contactInfo Optional
Sets the list of target email address to send a message to and their field names. The object should follow this JSON: 
    {
      "contacts" : [ {
                         "firstName": "John",
                         "lastName": "Smith",
                         "emailAddress": "john.smith@home.com"
                   }],
      "fieldNames" : ["firstName", "lastName", "emailAddress"]
    }
Field Detail
jobId
